Date: 2026-04-12

Preflight build used an isolated target dir per contention rules:

```sh
CARGO_TARGET_DIR=/tmp/fireline-w13 cargo build --bin fireline --bin fireline-streams
```

Result: build succeeded. All `run` scenarios below used:


## Summary

- Passed: 1, 2, 7, 8
- Failed: 3, 4, 5, 6

Main findings:

- `run` works against a valid harness spec and tears down cleanly on both `SIGINT` and `SIGTERM`.
